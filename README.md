# 🌍 Worldwide Energy Consumption Analysis using SQL

## 📌 Project Overview

This project analyzes worldwide energy consumption, production, GDP, population, and CO₂ emissions using SQL. The objective is to extract meaningful insights from global energy data using advanced SQL queries.

## 🛠️ Technologies Used

- MySQL
- SQL
- Joins
- Aggregate Functions
- Views
- Common Table Expressions (CTEs)
- Window Functions

## 📂 Database Tables

- country_3
- population_3
- gdp_3
- production_3
- consum_3
- emission_3

## 📊 Analysis Performed

- Top countries by GDP
- Total energy production and consumption
- CO₂ emissions analysis
- Population vs Energy Consumption
- Energy production by country
- Countries with highest emissions
- GDP and energy relationship

## 🚀 Skills Demonstrated

- SQL Query Writing
- Data Analysis
- Data Cleaning
- Joins
- CTEs
- Window Functions
- Business Insights
